## Moderation Toolkit

<CardGrid>
  <Card title="ğŸ“œ Warnings" icon="seti:book">
    - `/warnings list [member]` - Peek at someone's naughty list
    - `/warnings clear <member>` - Give someone a clean slate
    - `/warn <member> [reason]` - Give a gentle (or not so gentle) reminder to behave
  </Card>

  <Card title="â³ Timeouts" icon="seti:clock">
    - `/timeout <member> [reason]` - Put someone in a time-out corner
    - `/untimeout <member> [reason]` - Let them out of the corner
  </Card>

  <Card title="ğŸ‘¢ Kicks & Bans" icon="seti:error">
    - `/kick <member> [reason]` - Show someone the door (they can come back)
    - `/softban <member> [reason]` - Kick + delete messages combo move!
    - `/ban <member> [reason]` - The ultimate "goodbye" button
    - `/unban <member> [reason]` - Everyone deserves a second chance, right?
  </Card>

  <Card title="ğŸ§¹ Clean-Up Crew" icon="seti:broom">

    - `/purge bots [amount]` - Sorry, fellow bots!
    - `/purge links [amount]` - No more sneaky links!
    - `/purge token <token> [amount]` - Zap specific words or phrases
    - `/purge user <user> [amount]` - Clean up after specific troublemakers
