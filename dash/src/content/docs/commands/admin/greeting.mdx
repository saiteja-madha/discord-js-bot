# âœ¨ Greeting Messages

## Greeting Messages

<Aside type="tip">
  Available Variables:
  - {'{server}'} - Server name
  - {'{count}'} - Member count
  - {'{member:name}'} - Member's name
  - {'{member:mention}'} - Member mention
  ... and more!
</Aside>

<Tabs>
  <TabItem label="Welcome">
    Configure welcome messages:
    ```
    /welcome channel:#welcome
    /welcome status [on|off]
    /welcome desc content:"Welcome {'{member:mention}'} to {'{server}'}!"
    /welcome preview
    ```
  </TabItem>
  
  <TabItem label="Farewell">
    Configure goodbye messages:
    ```
    /farewell channel:#goodbye
    /farewell status [on|off]
    /farewell desc content:"Goodbye {'{member:name}'}!"
    /farewell preview
    ```
  </TabItem>
</Tabs>
